# DLP_Lab4

## How to use 

Before you run this code, you must install required library first, then execute code.

```
    $ source install.sh
    $ DLP_Lab4
    (DLP_Lab4)$ python3 main.py
```

## Use others setting 

you can choose network parameters by yourself.

```
    (DLP_Lab4)$ python3 main.py -h
```
