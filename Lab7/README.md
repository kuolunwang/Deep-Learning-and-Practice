# DLP_Lab7

## How to use

Before you run this code, you must install required library first, then execute code, take CGAN for example.

```
    $ source install.sh
    $ DLP_Lab7
    (DLP_Lab7)$ python3 main.py
```

## Use others setting 

you can choose network parameters by yourself.

```
    (DLP_Lab7)$ python3 main.py -h
```
